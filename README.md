# E-Wallet Backend

## 📌 Overview
Backend service for the E-Wallet system built with Spring Boot.

This system handles authentication, wallet management, transactions, and security features such as JWT and OTP verification.

---

## 🔗 Related Project
- Frontend: https://github.com/hoanganhquan29/ewallet-frontend

---

## 🛠 Tech Stack
- Java 17
- Spring Boot 3
- PostgreSQL
- Redis
- JWT Authentication

---

## 🔐 Security Features
- JWT Authentication (15 min expiry)
- OTP verification (2FA)
- AES-256 encryption
- Rate limiting (Bucket4j)
- Audit logging
